# Sniper Shoot

Sniper Shoot is a target shooting game where players test their accuracy and precision.

## Features
- Long-range sniper shooting
- Zoom system
- Score system (100, 80, 60, 40, 20)
- Limited ammo
- Bronze to Diamond rank progression
